# NPanel User Web

Modern user portal for NPanel, covering account access, subscriptions, payments, documents, tickets, and traffic visibility.

[Website](https://npanel.dev/) | [Telegram Channel](https://t.me/mynpanel) | [Telegram Group](https://t.me/NPanelChat)

[Chinese](./README.zh-CN.md) | [Repository](https://github.com/npanel-dev/NPanel-frontend)

</div>

## Local Development

```bash
git clone https://github.com/npanel-dev/NPanel-frontend.git
cd NPanel-frontend
bun install
bun run dev --filter=NPanel-user-web
```

Open <http://localhost:3000> to preview the user portal.
