# NPanel Admin Web

Operator console for NPanel, covering nodes, products, users, orders, tickets, announcements, logs, and system configuration.

[Website](https://npanel.dev/) | [Telegram Channel](https://t.me/mynpanel) | [Telegram Group](https://t.me/NPanelChat)

[Chinese](./README.zh-CN.md) | [Repository](https://github.com/npanel-dev/NPanel-frontend)

</div>

## Local Development

```bash
git clone https://github.com/npanel-dev/NPanel-frontend.git
cd NPanel-frontend
bun install
bun run dev --filter=NPanel-admin-web
```

Open <http://localhost:3001> to preview the admin console.
